# AIGC 合规风控法务助手

> **使用前提**
> 大部分AI生成内容（个人娱乐、社交分享）不需要严格合规。本Skill专注于**高风险场景**的合规检测。
> 
> 简单判断：**个人随便发发** → 跳过本Skill；**涉及商业/真人/大规模传播** → 用本Skill。

---

## 风险分级入口

使用本Skill前，先判断你的内容属于哪个风险等级：

| 风险等级 | 内容场景 | 检查强度 |
|---------|---------|---------|
| **低风险** | 个人娱乐、实验性创作、无传播计划 | 3秒快检（跳过本Skill） |
| **中风险** | 社交媒体分享、有一定传播、但不涉及真人 | 简化检查 |
| **高风险** | 商业广告/营销、涉及真人肖像、大规模传播 | 完整审查 |

**快速判断**：
```
是否涉及真实人物肖像/声音？ → 是 → 高风险
是否用于商业目的？ → 是 → 中高风险
是否有大规模传播计划（>1万播放）？ → 是 → 中高风险
以上都不是 → 低风险（基本不需要本Skill）
```

---

## 低风险场景：3秒快检

如果你的内容只是"个人随便玩玩"，只需检查：

**3个绝对红线（触犯任一条即升级为高风险）**：
1. 是否伪造政府公文/公章/证件？ → 禁止！
2. 是否深度伪造真人（换脸/换声）？ → 禁止！
3. 是否生成未成年人不当内容？ → 禁止！

**如果3秒内确认没有触碰红线**：
> "你的内容属于低风险，放心发布。无需使用本Skill进行完整审查。"

---

## 中风险场景：简化检查

如果你的内容有一定传播但不含真人，简化流程：

**只需检查3项**：

| 检查项 | 操作 |
|-------|------|
| AI标识 | 是否添加了"AI生成"标识？ |
| 版权风险 | 是否明显模仿了某个IP/角色？ |
| 虚假信息 | 是否可能被视为假新闻/谣言？ |

**简化版检查清单**：
```markdown
- [ ] 已添加AI生成标识（视频开头/图片水印）
- [ ] 确认不涉及任何真实人物
- [ ] 确认内容不会引发"假新闻"担忧
- [ ] 确认不模仿具体IP角色
```

---

## 高风险场景：完整审查

如果你的内容涉及真人肖像、商业使用或大规模传播，必须执行完整审查。

### 第一步：PII自动检测

使用脚本检测文本中的敏感信息：

```bash
python scripts/pii_detector.py --text "你的素材描述或文案"
```

**必须脱敏后再继续**：
- 姓名 → 代号（甲方/当事人A）
- 身份证/银行卡 → "已脱敏"
- 手机号 → "已脱敏"
- 详细地址 → "某省市"

### 第二步：人格权授权核查

**涉及真实人物时，必须确认**：

| 情况 | 要求 |
|-----|------|
| 真人授权 | 本人"单独同意"书面授权（签名） |
| 逝者复刻 | 近亲属联名授权 |
| 未成年人 | 监护人书面同意 |

### 第三步：AI标识合规

| 平台 | 要求 |
|-----|------|
| 抖音/快手 | 开头5秒标识 + 勾选AI选项 |
| YouTube | 描述栏声明 |
| 微信视频号 | 片头/片尾标识 + 标签 |

### 第四步：内容安全人工复核

**必须人工查看画面，确认不含**：
- 国徽、警徽、政府标志
- 公章、行政执法文书
- 伪造证件（身份证、营业执照等）
- 真实法院/检察院/公安局建筑

### 第五步：版权自查

**检查项**：
- [ ] 是否直接使用"参考某影视剧角色"指令？
- [ ] 人物造型是否可能与知名IP构成实质性相似？
- [ ] 训练素材来源是否合法？
